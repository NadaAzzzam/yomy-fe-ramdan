# iPhone Screenshots

Place your iPhone screenshots in this folder with the following naming convention:

## 6.7" Display (iPhone 14 Pro Max, 15 Pro Max, 15 Plus)
**Required - Priority 1**
- Portrait: 1290 × 2796 pixels
- Landscape: 2796 × 1290 pixels
- Files: 
  - `iphone-6.7-01-home.png`
  - `iphone-6.7-02-quran.png`
  - `iphone-6.7-03-azkar.png`
  - `iphone-6.7-04-prayer-times.png`
  - `iphone-6.7-05-ramadan.png`

## 6.5" Display (iPhone 11 Pro Max, XS Max, 11, XR)
**Optional but Recommended**
- Portrait: 1242 × 2688 pixels
- Landscape: 2688 × 1242 pixels
- Files:
  - `iphone-6.5-01-home.png`
  - `iphone-6.5-02-quran.png`
  - etc.

## 6.1" Display (iPhone 14, 14 Pro)
**Alternative Option**
- Portrait: 1179 × 2556 pixels
- Landscape: 2556 × 1179 pixels

## 5.5" Display (iPhone 8 Plus, 7 Plus, 6s Plus)
**Optional - Legacy Support**
- Portrait: 1242 × 2208 pixels
- Landscape: 2208 × 1242 pixels

## Screenshot Guidelines

### Content Recommendations
1. **01-home**: Main dashboard showing prayer times and Ramadan countdown
2. **02-quran**: Quran reader interface with Arabic text
3. **03-azkar**: Daily azkar with counter/list
4. **04-prayer-times**: Prayer schedule with notifications
5. **05-ramadan**: Iftar/Suhoor timetable

### Design Tips
- Use actual app content (not lorem ipsum)
- Add descriptive text overlays in Arabic (primary) and English (secondary)
- Show the app in use with realistic data
- Maintain consistent lighting and color scheme
- Include device status bar or use clean screenshot
- Consider adding device frames for professional look

### Tools for Screenshots
- **Xcode Simulator**: Built-in screenshot tool (Cmd+S)
- **Fastlane Snapshot**: Automated screenshot generation
- **Screenshot Creator**: https://www.screenshot.rocks/
- **Mockuphone**: https://mockuphone.com/
- **Figma/Photoshop**: For adding captions and frames

### File Format
- Format: PNG (24-bit)
- Color profile: sRGB or Display P3
- No transparency
- Maximum file size: 8 MB per screenshot

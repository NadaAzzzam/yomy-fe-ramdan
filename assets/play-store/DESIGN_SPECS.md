## ‚úÖ Export Settings

### Photoshop
```
1. Layer > Flatten Image
2. File > Export > Export As
   - Format: PNG
   - Quality: Maximum
   - Metadata: None
   - Color Space: sRGB
   - Uncheck "Transparency"
3. Save as: feature-graphic.png
```

### Figma
```
1. Select frame
2. Export settings (bottom right)
   - Format: PNG
   - Scale: 1x
   - Uncheck "Include 'id' attribute"
3. Export
```

### Canva
```
1. Share button > Download
2. File type: PNG
3. Quality: Standard (or High)
4. Download
```

## üí° Pro Tips

1. **Keep it simple** - Don't overcrowd the graphic
2. **High contrast** - Ensure text stands out from background
3. **Test readability** - View at thumbnail size
4. **Brand consistency** - Match your app's visual style
5. **Cultural sensitivity** - Use appropriate Islamic imagery
6. **A/B test** - Try different versions and see what performs better
7. **Update seasonally** - Refresh for special occasions
